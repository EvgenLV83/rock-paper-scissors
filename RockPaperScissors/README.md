# Камень-Ножницы-Бумага

Простая консольная игра на C#.

## Как играть
1. Запустите программу.
2. Введите:
   - `к` для **камень**,
   - `н` для **ножницы**,
   - `б` для **бумага**,
   - `в` для выхода.
3. Компьютер сделает случайный выбор.
4. Смотрите результат и счёт.
